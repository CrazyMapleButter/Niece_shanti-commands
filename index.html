Niece_shanti's Commands

Here is the list of available commands for tim's twitch bot Niece_shanti:

  • !discord - Sends the link to join the discord server.
  • !schedule - Sends Tim's stream schedule.
  • !rules - Sends the stream rules.
  • !lurk - Run this command when you're lurking.
  • !tmaple [message] - Send a message to Maple through tts and on discord (or just annoy Maple).
  • !8ball [question] - Ask the bot a question and it will respond with an answer similar to an 8-ball.
  • !hug [user] - Hug a user in the chat.
  • !monster - Drink a monster and it shows the number of monsters drank.
  • !giftmeasub - Do you really think this is going to work?
  • !petshanti - You can pet shanti.
  • !wands - sends the link to tim's onlyfa- I mean onlywands.
  • !whatishappeninginchatrightnow - send this when you have no idea what's happening in chat.
  • !timtime - This will tell you what time it is for Tim.
  • !annoymaple - This will DM maple 100 times on discord through timbot's discord account.
  • !annoyhero - This will DM hero 100 times on discord through timbot's discord account.
  • !jumpscaremaple - This will play a jump scare audio file for maple.
